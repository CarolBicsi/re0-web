# 『菜月·昂和文森特·埃布尔克斯』

挥出拳头的瞬间，就是最后悔自己变小的瞬间。

比起全员能够团结一致地跨越的剑奴孤岛的惨剧，比起以古斯塔夫为己方的「斯帕鲁卡」，比起和塞西尔斯的月下问答的时候，这一瞬间，比起小身体更大的身体，应该会稍微被狠狠地揍了一顿吧。

「你……！

年轻的昂的拳头打在左边的脸颊上，被血弄脏了脸的皇帝瞪大了眼睛。

他的视线在脚下，看着落地失败而滚到地板上的昂。然而，即使他的表情令人惊讶，也没有愤怒。比起屈辱，惊愕更强。

瞪着他的皇帝，在他眼前跳起身体，狠狠地批评。

不舍命，真是个骗子。

不过，我不想在推销语中说出一句话，在这里表现出一种软弱的心情。

对于这个随口说的皇帝，不管是语言还是拳头，都要反其道而行之。

「你们说得对。我觉得这是不可能的。我自己也不知道自己是以什么样的标准行动的。」

皇帝咬住嘴唇，瞪着自己，皇帝皱起了眉头。

但是，不管怎么想都不会明白。眼前的男人是皇帝，另一个男人只是没有职务的一兵卒。

但是，这两者给昂造成的伤痕，所指出的伤痕是一样的。

「我想优先保护喜欢的人。这是我诚实的心情。但是，即使是讨厌的人或者不认识的人，如果在眼前危险的话，我……」

托德说昂选择了一个因主观好恶而帮助的对象。

埃布尔说昂没有挑食，而是选择了无轨拯救的对象。

这两者都是正确的，两者都是正确的，所以是错误的。

因为--

「可是，这样不行吗……？」

大声疾呼踏着地团太的昂，皇帝睁大了他的黑眼眸。
