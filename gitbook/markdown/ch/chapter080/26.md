# 『毒公主.』

------

那是佩特拉・雷蒂一巴掌打在菜月・昴的脸上，弗洛普・奥克尼尔又发表了爆炸言论，导致米迪娅姆・奥克尼尔也混乱起来的时刻。

"果然，他们好像没有等到我们的到来呢。"

佩特拉咬着烟斗，抬起脸，城邦国家最强的忍术师沮丧地嘀咕道。

赫鲁贝尔，一个放松的肩膀让肩膀更加无力的狼人，坐在流动的夜景中，在龙车的屋顶上静静地警戒。

刚才，连环龙车本身就成为袭击目标。

多亏了赫鲁贝尔前面的孩子们提前发现了伏击，才摆脱了危险，但这种巧合并不常见。赫鲁贝尔的任务就是不让这种事发生。

「之后，那个恶心的傲慢家伙一直盯着我们。我也得挽回一下。」

这位少女是他认识的亲戚的女儿，为了这次去帝国的旅行，付出了巨大的代价。

赫鲁贝尔作为伴行者和城邦国家的使者，为了获取这个角色而努力。大概会启发商人灵魂的商机潜藏其中，但实际上，她的目的是寻找一个失踪的朋友。

这个女孩不想表现出弱点。当然，她肯定不会说出口。

"所以，老朋友们能帮上忙的话就好了"

说着，赫鲁贝尔缓缓站起来。

将多台龙车连结在一起，借助众多地龙一口气拉动的连环龙车——展现了帝国典型技术「避风的加护」的无厘头，真是贪婪至极。

即便如此，赫鲁贝尔也不习惯这种不自然感。

风景在流动，车轮在咬住崎岖的道路，却感受不到风的摇摆。卡拉拉吉地区地龙的数量较少，兽车较为普遍，摇晃和速度都很熟悉。

而谈到不习惯的不自然感——

「虽然动起来的僵尸也够可怕的……飞龙也会变成『僵尸』啊」

赫鲁贝尔一边将手指伸入袖间，一边懒散地挠着肚子，抱怨道。

透过斜着瞪着夜空的眼睛，映入眼中的是那巨大坚毅的身躯和展开的翅膀上满是痛苦龟裂的，变为僵尸的飞龙——「屍飞龙」的群体。

厚重得像覆盖住星星一般的黑云群，慢悠悠地向连环龙车靠近。

「——麻烦的是，里面有一个家伙啊」

赫鲁贝尔如此评价，似乎混在屍飞龙群中。

感知到这可怕的敌意，拖着连环龙车的地龙们也开始感到紊乱。

即使地龙被认为是忠实的人类朋友，但它们仍然没有完全失去野性。面对逼近的威胁，它们也会感到危机感并且会害怕。作为一种发明，连环龙车确实很了不起，但这种危机总是存在的。地龙本来就是一种具有高共情性的生物。一只害怕，恐惧就会迅速传播：

「——啊啊！！」

赫鲁贝尔刚刚担心完，就听到了划破黄昏的尖叫声。

然而，这不是迫近的威胁使地龙们慌张的致命声音。

尖叫的源头是领头车厢，数十只地龙排成一列拉动连环龙车，其中一只漆黑的地龙发出的叫声。它扮演着最重要的角色。

「哦哦，真是个有血性的家伙呢」

在黑色地龙的机智下，本来要遍布地龙们的恐惧被阻止了传播。

因此，害怕的地龙放弃了任务，最糟糕的情况——连环龙车空中分解的事故得以避免。正当赫鲁贝尔感到赞叹之际，他突然注意到了一件事。

「哎呀，那只黑色地龙是雌性。还真是有血性，但也有少女气啊。」

赫鲁贝尔不寻常地睁大了眼睛，微笑着说：「咔咔」。

笑声还在持续，作为狼人的隐者，他迈开轻松的步伐跑向竜车的外面——迈向空无一物的空间，向着天空跳跃。

迎击。为了避开连『礼赞者』都承认为棘手的强敌。

那个强敌——

"喂喂喂喂，那是什么啊！？"昴的声音倒放地传来，他贴在通道窗框上，看向窗外的景象。

这时候他已经与雷姆和路伊——不对，是丝碧卡——结束了客室里的讨论，与在担心中等待他们的爱蜜莉娅们合流，正在做报告的过程中接到了佩特拉的平手。

"这次是我和奥托的分数哦！"他受到了一次物理上的惩罚，咬紧牙关承受着她们的关心、以及即将继续下去的担忧，并巩固自己的立足之地。

突然，像击碎夜空一般的咆哮声响起，昴等人贴在窗上。

"那个是黑色飞龙……不对，比飞龙还要巨大！"

"那个大小，不是飞龙可以媲美的。毫无疑问，应该是龙级别的。而且……"

"——那个龙，竟然有三个头！"

昴和碧翠丝挤在窗边，而同样贴在窗户上的爱蜜莉娅尖叫道，不仅昴，连异世界的人也能认出它异形的威风。

在夜景中，一条巨大的黑龙在空中飞行，摇晃着它庞大的身体和三个头颅。它浑身裂开并展开的大翅膀，在夜空中发出三对金色眼睛的存在证明它的真实身份。

「僵尸龙......！！」

「这是有三个头的黑龙......难道是『三头』巴尔格伦？」

「你知道吗，姐姐！」

浑然不顾窗玻璃，昴三人转身回头看着身后的拉姆。拉姆抱着自己的手臂，用淡红色的眼睛瞪着外面的龙，

「以前，在帝国和王国交界的都市上横行霸道的邪龙。」

「果然是这样！有着这个绰号的独特怪物阿塔一定非常强大......」

「几年前的『亚人战争』之后，把商业都市夷为废墟的龙。还有......」

「还有什么？还有其他什么吗？」

「——。是上一代的家主，罗兹瓦尔大人的祖母在与它战斗中失去了性命。」

拉姆迟疑片刻后，昴三人倒吸了一口冷气。

当得知身边亲人成为祭品时，敌人的威胁感立刻变得更加真切。尤其是罗兹瓦尔的祖母，她肯定不会是个弱者。

如此惊人的邪龙，虽然现在先被赫鲁贝尔压制住了……

「那个，那边的狼人们是谁？」

坦萨一下子朝着昴突然凑过来，被窗外的攻防情景所吓了一跳。

她展现出的惊讶表情，看起来似乎错了又不错。

将正在战斗的赫鲁贝尔称为「那边的狼人们」，她的表达是正确的。

因为……

「赫鲁贝尔大概有三个人吧……」

「嗯，不，是四个人，昴。其中一个还在龙的背上，似乎想削掉它的翅膀」

「这种情况下，确切的数字不重要吧……」

凭借号称是都市国家最强的忍术，赫鲁贝尔在一瞬间就消除了斯芬克斯的致命攻击。光是这一点，已经足以证明他的名声。

而现在昴他们所见到的攻防情景，也足以证明他们对赫鲁贝尔的评价是正确的。

没什么好说的。赫鲁贝尔正如忍者一般分身行动，与邪龙进行着空中战斗。

在坦萨没有与赫鲁贝尔相识的情况下，由于他们表现出的高超战斗技巧，忍术组织「礼赞者」以为他是这个城市国家最强大的存在，这也不无道理。

无论如何——

「呜啊！」

「我觉得邪龙不是我们唯一的敌人。大家应该合流在一起才好吧？」

雪白丝和拉姆手牵手，高声提议合流。

本来是希望在更平静的情况下与大家分享有关雪白丝的事情的，但是既然情况发展了，就不能奢求那么多了。

「对，没错……现在就去大家那里！埃布尔他们也应该行动了。立刻……呃！？」

「昴！？」

正当他打算按照拉姆们的建议移动到别的地方时，

突然，昴被炽热的疼痛打断了话语，他弯下身去，爱蜜莉娅支撑住了他。

但是，不仅仅昴一个人感受到了那种不适。

「……没事吧？」

「竟然要你来支撑我，真是羞耻啊，鹿人姐妹……」

「竟然连碧翠丝殿下也……巴鲁斯，发生了什么？」

和昴一样，碧翠丝突然感到剧痛而晃晃悠悠，不满地向支撑住她的坦萨抱怨。

拉姆听到昴的回答「我、我不知道」，惊讶地问道：「突然间，胸口疼起来了……啊！？」

说着，昴把自己的衣领往下拉，瞪大了眼睛。

爱蜜莉娅和拉姆见到和昴一样的东西，也皱起了眉头。

眼前所见是一个红色的伤疤，像肿胀的蜜蜜一样——而且这不仅仅是一个普通的伤疤。

「这是什么……是谁的，眼睛的形状？」

「啊，果然你也这样看到了吗？这是眼珠子的标记……难道说，碧翠丝！你那边……」

「……贝蒂的娇嫩皮肤上，也有同样的标记。」

昴的衣领处有一个和萌芽掌一样大小的标记——是一种变形的、画眼睛的图案。

而碧翠丝的身体上也刻有同样的标记，他们俯身看着她展开的裙子的衣领，坦萨深深地点了点头。

「不过，只有我和碧翠丝吗？其他人没有什么不同吗？」

「太淫荡了。」

「只是纯粹的担心！难道不是因为这种状况和这个髻钗，才说出这种淫荡的话吗！？」

尚在持续疼痛着的标记，昴确认了其他人身上没有同样的标记，又用手轻抚着它，皱起了眉头。

在这种情况下出现了异常的萨伊茵。显然，这是昴他们与某个敌对势力的行动，但对方的意图不明。

或者说，昴和碧翠丝之间通过契约相连，所以对其中一方施加的某种力量也可能表现在另一方的身体上。

"不行，即使思考也无法理解。最好找一个有知识的人来诊断。总之，马上从这里……"

"啊啊唔！！"

"——不可以！"

改变主意，要提出从这里移动的建议时，又有人插了一脚。

丝碧卡和雷姆向昴他们并排靠近，一下子推开了他们。虽然是突然的举动，但这却是正确的选择。

——刚刚从昴他们正上方的屋顶突破而来的敌影的大剪刀，以暴力地撕裂昴他们之前所在的地方。

"边境伯爵！"

"大人！"

两人同时发出关切的声音，罗兹瓦尔苦笑了。

他们始终不愿意直接表达关心，也从未对罗兹瓦尔敞开心扉。但他们的担心毫无疑问是真实的，他们善良的天性不言而喻，也让人知道他们也是爱蜜莉娅阵营的成员。

――和罗泽瓦尔不同，我无法和他们接触。

「……啊，被标记了吗」

罗泽瓦尔咽下嘴角上扬的苦笑，打开了自己衣服的胸口。

佩特拉望着他疼痛的地方，那里红肿的伤口浮现出眼纹的形状。突然，罗泽瓦尔打开胸口，佩特拉的眼睛被掩盖的脸庞震惊了一下。

奥托看着那个印记，脸色阴沉地说道：「那是……」

「不可能，印记吗？」

「不可能中的不可能。这个，可能是被视为敌人的证明，或者是被定为目标的证明。无论哪种情况，都不是令人欢迎的。」

「敌人，或者是目标？」

奥托的目光落在佩特拉身边，她立刻明白了他的意思，摇了摇头。

从她的反应来看，奥托和佩特拉都没有这个眼纹。

虽然这本身是一个让人安心的事情，但是……

「那个眼纹，会有什么效果？」

「……可以考虑到会侵蚀生命，或者读取目标的思维。另外可能是为了始终追踪目标的位置。」

「特意从恶意的物品开始询问，嗯？那么，最有可能的就是用来追踪位置的印记了吗？」

「真是的，你们真是太出色了啊～」

罗兹瓦尔对奥托和佩特拉冷静地进行了广泛的推断，他闭上了眼睛。

这个判断毫无错误。能够侵蚀最危险的生命的类似诅咒的印记，无论如何也不可能远程施加，所以根本不用将其作为候选项。

当然，仅仅是定位位置的印记也足够令人头疼。

「或许，我们应该考虑另外的行动，以防止我们的行动被窃听。并且还有赫鲁贝尔大人的援助。你们跟我分开……」

「—不，应该已经太晚了」

低声回应后，奥托将佩特拉引到了自己身边。

佩特拉立刻瞪大了眼睛，她的眼前，一个巨大的身影破窗而入，闯进了连环龙车的通道。

那是一个全身覆盖着黑色铠甲，手持大刀，散发出凶恶气息的敌人——屍人。

「—！」

佩特拉咬住了尖叫，用手指指向敌人，奥托的怀抱中。

罗兹瓦尔一边赞扬那个试图在突发状况下先发制人的小女孩的勇敢，一边向前迈出一步，而在敌人挥起巨镰的前一刻，他用掌刀猛击了过去。

他将掌刀刺入对方兜上的缝隙，将手指插进对方眼窝中并念叨着。

「戈亚」

顷刻间，对方的头部被一团膨胀的火焰爆破所击中，同时装备被从内部炸裂。

尽管表现华丽，但罗兹瓦尔没有给予冲过来的敌人任何机会。他本以为这就是最好的方法，却后悔地发现自己大意了。

接连不断，从龙车的屋顶和通道上，同样装备的士兵们开始进入车内。

转眼间，连环龙车本身就变成了一个移动的战场。

「艾米利亚，快点过来！阿哥也跑！」

「其实我已经全力以赴在跑了，妹妹！」

米迪娅姆竭力地喊叫着，担任队长的他挥动着双剑击退敌人的进攻。

野刀和巨镰激烈地碰撞，他们趁机钻过缝隙，文森特和弗洛普在被他保护下成功抵挡住了敌人的猛攻。

从表面上看，这样的情景或许有些可怜，但其中的原因不仅仅是因为两人是受伤者和非战斗人员。

「皇帝殿下！没事吧？不会有人给你下毒吧！要是你违背了约定可麻烦了！」

「……哼。如果你这么担心的话，就别展现出太多野心。虽然可能会被认为是不敬」

「虽然你周围都是野心家，但世界上也有其他原因接近你的人，皇帝殿下！」

文森特一边扶着他的肩膀，一边嘴里嘀咕着荒谬的话，弗洛普却鼻子哼了一声笑了起来。

文森特面容扭曲了一下，仿佛有一股刺痛似的。突如其来的疼痛和红色疤痕，就像是作为文森特身体一部分的刺在腐蚀他。

紧接着，米蒂亚姆为了延缓敌人的袭击而停下来，文森特意识到自己正在经历着与以前相同，甚至更棘手的攻击。

那就是——

「那个胸口的伤痛……」

「那肯定是我兄弟帕拉迪奥.玛梅亚克的魔眼。这家伙，在『朝选之仪』中战死了，但似乎被重新复活了」

「魔眼……！它的效果是什么？」

「之前只能传递声音给被指定的目标，但现在死后的精确度提高了」

文森特一边回答弗洛普的问题，一边想起了自己曾经杀死的那个兄弟。

帕拉迪奥虽然在竞争帝位的兄弟中是属于难对付的一类，但他却因为自己的魔眼能力而自满，没有充分发挥其性能，这成为了他失败的原因。

没想到他死后竟会来纠正这个问题吗——。

「……不，就算死了也不是容易改变自己存在方式的。帕拉迪奥被命令执行这件事的，肯定有个人存在。」

「但是，如果是你的兄弟的话，帕拉迪奥这个人物也是皇族吧？譬如说就算是变成屍人，也不可能让这样的对手听从自己的话……」

「——大概你已经察觉了吧。如果帕拉迪奥选择顺从的话，那只可能是前任皇帝德赖森・佛拉基亚，或者是比他更高位的人，而他本人也会屈服。」

满足以上条件的人只有文森特自己，但考虑对手是屍人这一事实，他也排除在候选之外。

可靠的屍人候选人中，德赖森作为前任皇帝和实际上的亲生父亲浮现在思考的对象之中，同时也想到了行踪不明的妹妹作为毫无价值的条件。

但是，考虑到米迪娅姆所面对的敌人的装扮，实际上敌人只能是一个人选——

「——拉米娅吗。」

满足条件并带着帕拉迪奥的恶名兵器——「剪枝部队」，拉米娅・戈德温之外再无其他人会具备这样的实力。

在争夺皇位的「选帝仪式」中，文森特推定了三位兄妹中的一位作为假想敌，并紧紧抓住了被痛苦侵袭的胸膛。

就在此时——

「真是烦人啊，一群死尸！！」

一声粗野的咆哮声响起，刃光在文森特一行人前方肆虐而过。

仔细看去，穿窗而入的尸人们被从相反方向过来的士兵——那个有着卷曲黄发和眼罩的男子斩成了碎片，并化为尘埃。

紧接着，那名喘息粗重的士兵注意到文森特一行人后，

「嘿！你们，帮个忙！我妹妹在车里死掉了……」

「不，帮忙的应该是你才对。」

甜腻的声音再次掠过耳畔的瞬间，哥兹的巨锤向后挥了一下，正好击中了背后潜行的敌人的身体，伴随着陶器碎裂的声音，将敌人的身体斩成两截。

哥兹看着那飞散的碎块，喉咙一阵干涩。

被反射攻击击倒的敌人，消散成了尘埃，那是因为那张脸实在太不可能存在了。

因为刚刚这个刚才，它应该已经被哥兹亲手击败了。

「好罕见，居然说了这么聪明的话呢。」

「笨蛋……」

面对这个绝对不该发生的异常景象，握着锤矛的手颤抖了起来。

这是因为愤怒吗，还是其他的情感引起的呢，连哥兹自己都无法分辨。唯一确定的是，这是一种亵渎。

哥兹・拉尔冯，狮子骑士，对佛拉基亚帝国的亵渎。

「「——这是我们被困在过去与你们残存的我们之间的灭绝之战！」」

戈德温的拉米娅，曾经粉碎过的，向哥兹挤眉弄眼地笑着说道。

——手持无数「阳剑」的无数「毒姬」拉米娅・戈德温们。
